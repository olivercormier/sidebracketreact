# sidebracketreact

### Prerequisites

* Node v14.13
* sidebracketapi installed as described [here](https://github.com/olivercormier/sidebracketapi/blob/master/README.md)

### Installation 

* Open terminal
* Clone this repository
    * git clone https://github.com/olivercormier/sidebracketreact.git
* cd sidebracketreact
* npm install

### Run Server

* npm start
